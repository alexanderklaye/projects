This program is a simple litebrite that shows knowledge of the Model View View-Model, the use of Binding in C# applications, as well as C# coding skills.

The program can randomize the colors of the 50x50 tile board, as drag and drop functions on both the board and the list of available colours. The program is able to save and load board configurations held in .txt files.

It has a number of pre-made images saved alongside it
